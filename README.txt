Compañeros un poco de orden (convenciones):
============

A nivel de la vistas de django en relación a los templates:
Si devolvemos una lista de objetos, esta se debe asignar a una variable llamada object_list (obvio?, no, para nada)
Si devolvemos un objeto, para una vista de detalle, por ejemplo, este se debe asignar a una variable llamada object (obvio?, no, para nada)

Esta convención es usada en las vistas genericas de django y en casi todas las aplicaciones que reutilizamos.
